Built for real dictation

One menu bar tool, a few good choices, almost no clutter.

  • Original Chinese output in Simplified or Traditional Chinese
  • Translate spoken Chinese into English
  • Natural English polish for cleaner phrasing
  • Switch input sources: built-in mic, headset, USB mic, iPhone, virtual devices
  • Custom dictionary for names, brands, and common correction rules

Best for:

chat boxes, quick notes, bilingual drafting, and fast desktop dictation Designed for macOS. Source-first. Local-first.
